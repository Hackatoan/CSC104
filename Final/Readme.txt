This application is a final project assignment for the CSC104 course.

It functions as a simple music player

- Displays a main library of all available songs.
- Allows users to play songs directly from the library or from custom playlists.
- Users can create playlists and add songs to them.
- A persistent player bar provides controls for play/pause, next/previous, volume, shuffle, and loop.
- All playlist data is saved in the browser's local storage, so they will be available on subsequent visits.
- The interface is responsive and adapts to different screen sizes.
- Working Search functionallity
- Descriptive about and contact pages
- Working titles

All songs featured in this application are sourced from NCS.io (NoCopyrightSounds), providing royalty-free music for creators.

Album cover images are dynamically generated and provided by Picsum.photos.

Favicon is generated by Gemini and ran through favicon.io.
